Tentang Saya

Halo! Saya Azira Faradina, biasa dipanggil Zira. Saya lahir di Samarinda pada 24 Agustus 2006 dan saat ini berusia 19 tahun. Sekarang, saya sedang menempuh pendidikan di Universitas Mulawarman, Fakultas Teknik, Program Studi Informatika.

Saya memiliki ketertarikan di bidang Software Engineering, khususnya dalam pengembangan iOS. Cita-cita saya adalah bekerja di Apple, California sebagai Software Engineer untuk menciptakan produk teknologi yang bermanfaat dan digunakan oleh jutaan orang di seluruh dunia.
